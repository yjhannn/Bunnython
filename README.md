#Bunnython

--Usermode

 1. 예약 기능
  - 기존의 부킹 예약 어플처럼 호텔 예약을 진행할 수 있음.
 2. 객실 제어 기능
  - 객실의 출입을 실물 스마트 키가 아닌 스마트 폰 내의 가상 키로 출입할 수 있다.
  - 객실 내 조명이나 온도를 제어할 수 있다.
 3. 부가 서비스 이용
  - 룸 서비스를 어플로 간편하게 이용 가능하다.
 4. 여행지 추천
  - 사용자가 고른 여행지와 사용자의 현재 위치를 기반으로 최적 경로를 추천해주는 기능이다.
  - 자신의 후기를 남겨 다른 사용자들이 참고할 수 있습니다.
